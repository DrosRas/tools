USER LEVEL DOCUMENTATION FOR ARP SPOOFER


USER MUST HAVE BASIC UNDERSTANDING OF LINUX TERMINAL COMMANDS AND NAVIGATION
USER MUST HAVE BASIC NETWORKING KNOWLEDGE

BEFORE RUNNING THE PROGRAM USER MUST ENTER THE COMMAND :
echo 1 >> /proc/sys/ne/ipv4/ip_forward (AS SPECIFIED BY THE ARPSPOOFER PROGRAM TO FORWARD THE PACKETS TO TARGET MACHINE)

USER MUST PROVIDE ROUTER IP AND TARGET IP AS ARGUEMENTS WHEN RUNNING THE PROGRAM 
FOR EXAMPLE:   python3 arpspoofer.py 1.1.1.1 2.2.2.2 (TARGET MACHINE IP AND ROUTER IP IN A LOCAL AREA NETWROK)
